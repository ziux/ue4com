難度

不同的「難度」設置給予玩家或AI國家不同的加成，以調整遊戲的整體難度。如果設置為正常，沒有國家會得到加成。在簡單的情況下，AI對玩家更寬容，而在困難的情況下，AI則更具攻擊性。

幸運國家

決定是否有8個歷史，或者8個隨機，或者沒有幸運國家。

在地圖上表示未知之地

開啟或關閉戰爭迷霧。特別是在你自己發現前不想提前知道隨機新大陸長什麼樣時很有效。鐵人模式不可用。

使用動態省份名稱

當省份的文化組或者擁有者改變時，允許或禁止更改省份的名稱。如果該選項關閉，只能人為改名，系統不能對其進行改名。

顯示月度稅收

在地圖上以動畫文本的形式顯示月度稅收。

「有主」荒土上色

荒蕪省份將被塗上周圍主要控制者的顏色。

飛地使用地區名

飛地命名使用地區名（如以「瑞典剛果」代替「瑞典」。）

理念組選擇沒有限制

你可以選擇任何的理念組，不再受類別的限制。

幻想的隨機新大陸

控制在隨機新大陸出現幻想元素（如高度發達的原住民帝國）的幾率。

顯示開始介面

開關是否在一局新遊戲開始時顯示有關你的國家與所在地區的背景信息的開始介面。

多人遊戲

 此內容可能已落後版本，最後更新於1.29

允許玩家熱加入

如果開啟，玩家可以加入一場已經開始的遊戲，使其變成多人遊戲。在鐵人模式啟用時不可用。

允許多個玩家使用同一國家

如果開啟，多個玩家可以控制同一個國家。在鐵人模式啟用時不可用。

僅允許主機與觀察者存檔

如果開啟，只有主機和觀察者將能保存多人遊戲的存檔。I

僅允許主機暫停

如果開啟，只有主機能夠暫停遊戲。

Editable savegame

If on, players can manually edit saved game files between sessions.

Locked ledger

If on, players will not be able to open the ledger, preventing usage of it for seeing other countries' armies.

Limited ledger

If on, the amout of information given by the ledger is reduced.

Use victory cards

Enables od disables victory cards in the game.

Use age scoring

If on, scores and victory cards will be reset with every new age.

Use teams

If on, members in a team can only ally each other and not other players.

Free team creation

If on, every player can create a team in the lobby.

Prevent nation ruining

Blocks certain destructive actions by players that have lost a war as long as the truces after that war are in effect.

Allow province name change

Allows players to change names of provinces.

自訂設置

 此內容可能已落後版本，最後更新於1.29
只適用於DLC黃金國激活時。

自定義國家難度

這個選項決定玩家創建自定義國家時的可用點數。點數越多，難度越低。

點數: 50 - 100 - 200 - 400 - 800

省份基稅及人力

這個選項決定所有省份的基稅和人力。

正常值:

所有省份具有史實的稅收和人力值。

隨機值:

所有省份具有隨機的稅收和人力值。

同一值:

所有省份具有相同的稅收和人力值。(4-4-3)

國家

這個選項決定地圖上會出現哪些國家。

正常設置:

啟用自定義國家，這張地圖將由史實國家和已啟用的自定義國家組成。

史實設置:

禁用自定義國家，這張地圖僅由史實國家組成。

隨機設置:

這張地圖將由玩家自定義的國家和隨機生成的國家組成。

自訂設置:

這張地圖僅由玩家自定義的國家組成，其他省份都是未殖民省份。所有國家開局獲得一個殖民者。

允許向存檔中加入國家

允許向存檔中加入自定義國家。

隨機國家

這個選項決定了隨機自定義國家的名稱及旗幟樣式。

動態:

隨機自定義國家會得到動態生成的名字和旗幟。

史實:

隨機自定義國家會得到歷史上存在的國家的名字和旗幟。
