游牧风云

游戏模组：游牧风云
游牧风云
LOGO
别名/游戏中名称 Nomad Universalis
类型 时间线延长
作者 克尔苏加德K、厂公天下、法国球
版本 3.03威力加强版：胜利的草原对应1.15.1 版本
论坛/贴吧 游牧风云吧

游牧风云是欧陆风云一个模组。游牧风云有两个大版本游牧风云2（对应欧陆风云3）和游牧风云3（对应欧陆风云4），3代主要是由克尔苏加德K单独制作，后来也有法国球的参与。内容是1206年蒙古统一至现在的世界局势模拟，可单人和多人游戏（3代目前多人游戏尚不完善，谨慎使用）。

本页主要介绍游牧风云3。基于欧陆风云4的游牧风云3已发布多个版本。目前，游牧风云3已进入一个稳定版本，更新将以跟进版本、修复BUG和保持稳定性为主，以及添加大量新国家理念、事件和决议。下一个版本将是基于1.22的3.04；同时xxyxxyxxj仍在更新基于1.15的非官方版本（不在本页介绍范围内）。

配置的基本要求

最低配置要求：

操作系统：Microsoft Windows XP / Vista / Windows 7 / Windows 8 / Windows 10

处理器(CPU)：Intel 双核奔腾 2.2Ghz 或相同等级的AMD CPU（速龙双核之类）

内存(RAM)：2G 1066Mhz

显卡(graphics card)：支持DirectX9.0c、Vertex- and Pixelshader 2.0，显存最少128MB

推荐配置要求：

操作系统：Microsoft Windows XP / Vista / Windows 7 / Windows 8 / Windows 10

处理器(CPU)：Intel 奔腾 G2030 @ 3.00GHz 或AMD Athlon X4 640 3.00GHz

内存(RAM)：4G双通道 1600

显卡(graphics card)：GTX 630或HD6650

事实上，再多核心也没什么用，P社四萌只能调用单核运行，所以处理器频率越高越好，也许3Ghz的单核老奔四都比一些双核跑得顺溜...XP系统2G内存就够了，VISTA以上的2G内存会比较吃力，建议还是上4G吧。游牧打开后就吃了1G多的内存了...

内容相关

初衷

阿提拉、成吉思汗、帖木儿......奥斯曼人、莫卧儿人、满族人、英国人、俄罗斯人、日本人和纳粹帝国。所有建立起来的帝国都想要永远的持续下去——他们都失败了，没有任何一个国家能够跨越管理的困难，将世界融入一个帝国——但是，正是他们曾经努力去创造帝国的行动改变了之后的世界历史。历史有其必然性，但历史也有其偶然性，没有人知道明天会发生什么，但是历史会见证它，我们可以改变未来。在游戏里，我么可以完成这些人没有完成的事业，建立世界帝国。当然，在现实里，把天朝建设为不灭的世界帝国也是每个中国人应该有的梦想。地理因素能够隔绝时空的时代已经过去，新的时代里，不只有伟大的人才能创造伟大，游牧风云的世界就是这样的——没有不可能，用自己的智慧，帮助那些顺应历史潮流的国家发展壮大，或是让落伍于时代的国家跟上步伐，挽救这个国家于倾颓。

理念

国家理念，用这个中文的翻译正确的解释应该是国家民族的精神层面带给国家的要求。但是似乎三和四的版本中都不是这样的东西，更像是这个国家的基本国策。在游戏的源文件中，理念部分的英文是ideas，可以表示建议或是意见，严格来说称为理念是完全不合适的。政策，或是基本国策的翻译更为合适。

同时还应当注意，无论是民族心性还是国家政策，对于一个国家和民族的影响都是一把双刃剑。民族的心性将作为特色理念组的起始，而一部分细化过的理念将会有负面效果。这样更容易平衡，同时也反映事实，甚至能够让国家的特点更加鲜明。游牧风云3的新版本最需要的就是这些能够让每个国家都有自己的特色发展崛起路线，而不是仅限于原版纯游戏性但是却没有太多可玩的乐趣的理念组。

在其它的游戏，例如信长之野望中，所有的政策都或多或少的带走一定的负面效果，因为事实上不存在一个完全正面性的东西。举一个简单的例子：俄罗斯的冬天导致了其肥沃的土地因为缺乏热量而产量不好，导致了其发展水平几乎一直落后于欧洲，然而这却是对付外敌的恐怖阴影。而欧陆风云4中去除了欧陆风云3中大部分带有负面效果的决议等带有的负面效果，这部分是游牧风云3尚未完善之处。

历史性与游戏性

在欧陆风云4原有的游戏引擎基础上，游牧风云延伸了历史的主时间轴，并尽力用丰富游戏中的乐趣和大时代的历史感，其最初的设计理念即世界性帝国的崛起和衰落，这个话题是不曾改变的。那些轮番登台于世界舞台的霸权国家将很容易的建立自己的帝国，并快速取代前任霸主。这个是主要的历史时间轴。史诗性是史实向的主要体现。虽然原版的欧陆风云是模拟1444-1821年的历史，对之后或之前的历史还原不够好，但经过制作组的设定与试验，证明了欧陆风云3的环境也能很大程度模拟近现代的局势，而欧陆风云4的版本尚未完善。对于玩家来说，玩家需要的是在游戏中的乐趣，而不是去片面地追求历史性。

新增

国家

游牧风云的鼻祖是EU3时代的群雄世纪。游牧风云相对其则对历史部分进行了改善，作者根据历史资料的描述，除之前的达尔富尔和科尔多凡两个苏丹国外，还增加了撒哈拉部落国家希拉勒和苏莱姆（他们经常骚扰法蒂玛王朝和阿尤布王朝的沿海地区），以及在索马里的游牧部落阿法尔（17世纪他们灭亡了阿达尔苏丹国）。于之前版本中，原本应该已经分裂成古罗斯三大公国的基辅罗斯还是一个整体（虽然名义上是这样，但是事实上弗拉基米尔、 诺夫哥罗德和 基辅在13世纪已经是三个完全不同的国家），这里做了详细的修正。另外钦察各部，在群雄世纪上统一归属为库曼-克普恰克汗国，经考证还原了大概的钦察各部的版图，不再是统一的国家，而是多个游牧国家。

对于1821年之后的世界，则增加近现代国家，并为一些国家设立了特色理念、决议等。为了照顾喜欢玩中国的玩家，加入了太平天国，还有建立中华帝国、军阀混战的事件。

剧本

除了原版欧陆风云4带有的剧本外，新增了大量史实事件的开局剧本，例如蒙古崛起、日本战国、太平天国、一战二战、军阀混战以及现代等多个剧本，让玩家可以随意选择，是要选择尼西亚恢复罗马荣光，阿拔斯统一伊斯兰世界，南宋小朝廷踏破贺兰山缺，亦或是用太平天国推翻满清王朝，甚至用 格鲁吉亚逆袭强大的战斗民族，由玩家决定！

原本在游牧风云2中最后的几个基于鲁鲁修的未来架空剧本，在基于欧陆风云4的游牧风云3里不再会出现。

特色系统

意识形态系统：新的意识形态有两个组，左翼和右翼

左翼：社会主义、共产主义

右翼：自由主义、保守主义、法西斯主义

国名转换系统

已成功模拟政体影响国名与国旗的系统，利用事件强制使部分国家改变政体后切换tag或是到一定时间改换国名。

动态地名

修复了原汉化版动态地名乱码或是不翻译问题，汉化了所有动态地名文本并修复了一些别的汉化错误。

理念组

平衡了原版的理念组并添加了大量新的理念组。

种地

添加了大量建筑及省份建筑槽，并且三项点数十分富余。这也导致了扩张十分困难，尤其在后期。

尚未完善的地方及明显的BUG

1.宗教、意识形态的特色决议。原版1.15只有基督教、伊斯兰教等拥有大量宗教特殊系统及决议，最近的版本已经完善了现代意识形态的系统，还有很多比如科普特正教、儒教、犹太教、印度教、原始宗教仍处于白板状态，但应该会慢慢补充。

2.自定义国家继承人名字库、顾问、外交官、将领全为Adam。

3.少部分历史文件错误。（比如比萨的政体历史上13世纪应为商业共和）

4.部分海域有探险队时会卡死不动直到探险家死亡（在3.03威力加强版中修复了一些海域的此问题，但该问题仍然会在部分海域出现）。

下个版本3.04首要计划更新的内容

1.尽可能贴近原版，理念可能改动，但理念加成的修正不会变态。1444-1821玩上去就要像原版一样没区别。

2.在简单、贴近原版、符合游戏性原则的基础上，增加新内容：更丰富的决议、事件和能够决定历史走向的内容。

3. 能够更多的还原历史事件，最终的目标暂定是从1206年挂机看海的话，完全能够通过事件和决议基本模拟到2017年的世界版图。

常见的疑难问题

1. Q:为什么我运行游戏时出现d3dx9_XX.dll丢失，无法运行游戏？

A：这是系统DirectX 9.0c缺少该dll，下载相应的dll放进系统盘的system32文件夹里即可（32位系统）。

2. Q：为什么玩着这么卡？

A：由于游牧风云的数据量比原版多了很多，运行速度减慢是不可避免的。和原版相同的一点是一个档随着游戏内时间的推进存档数据越来越多必然会导致越来越卡，一个缓解的办法是尽量开启早期剧本游玩并且达到目标后封档而不是从1206年一直玩到2017年。最根本的解决办法是使用单核处理性能更强的处理器，更高频、稳定的内存，还有固态硬盘。

3. Q：为什么启动后国家旗帜全黑/存档0kb？

A：这些问题是欧陆风云4原本就有的问题，和游牧无关。请自行欧陆风云4吧搜索解决方法。

4. Q:非西方国家要不要西化？

A：在游牧风云里平衡了各个科技组不同时代的科研惩罚以模拟历史进程，因此无需西化（西化反而破坏游戏平衡）。具体还是由玩家自行决定。

5. Q：怎么不能成立不列颠法兰西联合帝国、神圣布列塔尼亚帝国、中华联邦？

A：游牧风云3中已经移除架空国家。

6. Q：看不到最下面的理念组。

A：分辨率不足，如果是笔记本建议换成台机。
