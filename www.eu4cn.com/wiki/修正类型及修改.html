修正修改

本页面讲述的内容长期有效。
（重定向自修正类型及修改）

本章主要介绍各种不同的修正及修改方法。

概括地说，首先要写一个修正，它包括修正列表中的某些功能项目和数值；然后再用某种方法（事件、触发等）把写好的修正赋予目标国家或者省份。所以说，修正不是指令。

静态修正

静态修正只会作用于某些特定的国家或省份。由于它属于硬编码，这意味着不能添加更多新的静态修正，只能更改它们对应的修正效果。

静态修正的内容可以在/Europa Universalis IV/common/static_modifiers/*.txt处找到。

一个可以对整个国家产生影响的静态修正，例如专制度，它与国家的专制度直接相关，具体修正如下。 

absolutism = {
	administrative_efficiency = 0.4
	discipline = 0.05
	core_decay_on_your_own = -0.5
}

限时修正

限时修正会在特定时间内对特定国家产生一些特别的效果。由于它属于硬编码，这意味着不能添加更多新的限时修正，只能更改它们对应的修正效果。本地化文件中只有名称：<name>:。

限时修正可以在/Europa Universalis IV/common/timed_modifiers/*.txt中找到。

限时修正的一个例子： 

place_relative_on_throne_not_regency = {
	liberty_desire = {
		value = 25
		yearly_decay = 1
	}
}

关系修正

关系修正可以用来修正外交关的变化, 即关系变化的程度、时间和最大值。一般应用于事件、决议，或者用于add_opinion内。

关系修正可以在/Europa Universalis IV/common/opinion_modifiers/*.txt中找到。百科列表在观点修正页面。

关系修正的一般格式如下： 

<name> = {
    opinion = <int>                 # 关系增加数。可为负数。
    min = <int>                     # 可选。此修正可作用的最小效果。
    max = <int>                     # 可选。此修正可作用的最大效果。
    max_vassal = <int>              # 可选。此修正对附庸国可作用的最大效果。
    max_in_other_direction = <int>  # 可选。此修正对派遣（外交官）国家可作用的最大效果。
    
    yearly_decay = <int>            # 可选。关系每年衰退的值。可为负数。
    months = <int>                  # 可选。此修正可维持的时间（月份）。
}

关系修正的名称需要在本地化文件中添加。本地化文件中只有名称：<name>:。

事件修正

事件修正是游戏里使用最广泛的修正，它常用于对国家或省份属性的修正。通常事件修正需要通过效果语句方能添加一个效果，如： add_country_modifier 、 add_province_modifier ，可以在效果页面看到所有可能添加的效果。

事件修正可以在/Europa Universalis IV/common/event_modifiers/*.txt中找到。百科列表在event modifiers页面。

事件修正的一个例子： 

the_proper_old_ways = {
	raze_power_gain = 0.20
	land_morale = 0.1
	global_trade_goods_size_modifier = -0.25
}

修正 名称 可以在本地化文件中添加或修改。

默认情况下，事件修正的图标将使用修正本身一个效果的图标。但是，如若是事件修正是被作用到一个省份，您可以使用指定的图片作为特定的图标。

一个带 图片 的事件修正： 

important_natural_harbor = {
	province_trade_power_value = 10
	naval_forcelimit = 1
	local_development_cost = -0.05
	local_sailors_modifier = 0.25
	picture = "province_trade_power_value"    #使用指定图片作为图标
}

你也能添加描述，如果事件修正被应用于一个省份。

 desc_<name>: "Description"

可用的图标被储存在 /Europa Universalis IV/gfx/interface/ideas_EU4 文件夹中。

贸易修正

贸易修正是指在特定贸易节点中可以改变国家的贸易能力的修正效果。在贸易节点范围内的省份可以使用add_trade_modifier添加省份修正效果。详见指令页面查看关于add_trade_modifier的详细内容。

一个添加贸易修正的例子： 

random_active_trade_node = {
         limit = {
                 continent = asia
                 NOT = {
                         has_trade_modifier = {
                                 who = ROOT
                                 name = merchant_recalled
                         }
                 }
         }
         add_trade_modifier = {
                 who = root
                 duration = 3650
                 power = -10
                 key = merchant_recalled
         }
}

修正的名称可以在本地化文件中修改。本地化文件中只有名称：<name>:。

触发修正

主条目：触发修正

触发修正（Triggered modifiers）是在国家或省份设置触发条件，达到条件后可以获得的动态修正。

触发修正可以在/Europa Universalis IV/common/triggered_modifiers/*.txt中找到。

注：在旧版本中，触发修正即全局触发修正，并不区分省份与国家。省份触发修正为1.24新增功能。

触发修正的一般格式如下： 

<name> = {
    # 此项决定修正是否在可触发修正窗口内可见。
    potential = {
        <条件>
    }
    
    # 触发修正所需的条件。
    trigger = {
        <条件>
    }
    
    # 可触发修正触发时获得的修正效果
    <修正效果>
}

修正 名称 可以在本地化文件中修改。格式有点特别：<name>:和desc_<name>:。

注：若非必要请不要加入过多的可触发修正。由于可触发修正本身的设置，它们会在每个月对每个国家进行检测是否达到触发条件。这意味着过多的可触发修正会导致游戏流畅度的降低，从而影响游戏体验。

省份触发修正

省份触发修正（Province triggered modifiers）是可以对省份产生直接影响的修正，属于动态修正。它们'必须用add_province_triggered_modifier应用于省份才能产生效果。有关 add_province_triggered_modifier 的所有详细信息，请参阅指令页面。

省份触发修正可以在/Europa Universalis IV/common/province_triggered_modifiers/*.txt中找到。

省份触发修正的一般格式如下： 

<名称> = {
    # 触发的修正是否在省份界面中可见的条件
    potential = {
        <条件>
    }

    # 触发修正所需的条件
    trigger = {
        <条件>
    }

    # 触发修正的效果
    <修正效果>
    
    # Province scope
    # 当触发修正的条件满足而开启时执行的效果
    on_activation = {
    
    }
    
    # 当触发修正的条件不满足而关闭时执行的效果
    on_deactivation = {
    
    } 
}

省份触发修正的名称需要在本地化文件中添加。格式和全局的触发修正一样：<name>:和desc_<name>:。


君主修正与人物特质

一般事件中通过add_ruler_modifier添加给统治者的修正，属于事件修正。一旦君主死亡，修正就会失效。

类似的，特质也可以看作是一种按照时间或其他条件自动赋予的人物修正。显而易见，统治者特质用于君主一家；将领特质只用于将领；AI特质只用于AI。在本地化方面，君主特质和将领特质的结构是一样的。

特质代码很容易就能修改成普通的事件修正，而且不需要改动本地化。由此成为一种没有扩展包的情况下的备选方案。
