修改派系

所有原版派系都可以在/Europa Universalis IV/common/factions/00_factions.txt里面找到。 你可以将派系添加到该文件，也可以自己创建一个新的文件

增加一个派系

要添加一个新的派系，请遵循这个格式: 

<faction> = {
    monarch_power = <type> # 手动提升该派系影响力所需要消耗的君主点数类型
    
    # 决定派系在界面上是否可见。 always = yes 是默认设定
    <triggers>
    
    # 可选，当掌权的派系改变时触发的标题和描述
    triggered_faction_name = {
        name = <new faction name>
        
        trigger = {
            <triggers>
        }
    }
    
    # 当派系掌权时提供的修正
    modifier = {
        <country modifiers>
    }
}

运用一个派系

要给一个国家添加一个派系, 需要将派系加入该国家政体的机制。为此，向政体增加以下代码在 /Europa Universalis IV/common/governments/00_governments.txt文件里 

faction = <faction>

此外，你还可以通过以下方法在脚本（例如事件）中启用和禁用派别: 

add_faction = <faction>
remove_faction = <faction>

派系影响力

创建动态的派系的国家作用域的修正. 它们遵循以下格式: 

<faction>_influence

你可以使用 add_faction_influence 效果来立即增加派系影响力. 

add_faction_influence = {
    faction = faction_tigers
    influence = 10
}

本地化

以下本地化用于处理派系： 

 <faction>: "Faction name"
 <faction>_influence: "Faction Influence"
 <faction>_FACTION_DESC: "Faction description"

界面

每个新的派系需要一个图形化的定义作为其图标和背景。它们遵循以下格式: 

spriteTypes = {
    spriteType = {
        name = "GFX_faction_bg_<faction>"	
        texturefile = "gfx//interface//faction_blue_frame.tga"
        transparencecheck = yes
    }
    spriteType = {
        name = "GFX_faction_<faction>"	
        texturefile = "gfx//interface//<faction icon>.dds"
    }
}

图标大小在90×90像素到110×110像素之间，具体这取决于你的风格。

你可以为派系影响力的修正 <faction>_influence 增加一个自定义的修正图标， 通过在/Europa Universalis IV/gfx/interface/EU4_ideas/里放置一个名为 <faction>_influence.dds的图片文件，这个修正图标不需要图形化定义。
