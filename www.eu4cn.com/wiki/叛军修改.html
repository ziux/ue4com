叛军修改

本页面所适用的版本可能已经过时，最后更新于1.25。
 此页面尚未完成
 需要完善以下内容： 部分项目的具体指标未明，需要补充

本条目介绍对叛军的修改。叛军的文件位于/Europa Universalis IV/common/rebel_types,你需要为每一种新的叛军创建一个新的文件。

一般地，在添加一种新的宗教之后，你往往需要在此地址中添加新的宗教狂徒类叛军。但是你也可以新建其他类型的叛军。

目录

  • 1 文件格式
  • 2 界面
  • 3 本地化
  • 4 参考资料

文件格式

叛军名称={
##基本属性
color={ R G B }请求验证具体效果
area = [nation/nation_rebel_tag/nation_religion]#叛军的活动区域。一般地，非分裂主义者叛军均为nation，即在爆发国范围内移动。分离主义者为nation_rebel_tag，也就是会在目标国家核心省份的地区间移动。
government= [monarchy/republic/theocracy/anti/any]#叛军所期望的政府类型，宗教类叛军填写theocracy(神权)请求验证具体效果
religion= <宗教> #叛军的宗教。应用于宗教狂徒。
defection = [culture/culture_group/religion/any/none]#叛军的叛逃目标，一般为none，分离主义者为nation_rebel_tag，少部分宗教狂徒为religion，即叛逃目标为信仰他们宗教的国家。请求验证具体效果
defection_delay= <int>#叛逃时间，以月为计算单位。
independence = [culture/culture_group/religion/any/none]#是否期望独立，非分裂主义者叛军可填写none。
unit_transfer = <boolean>#军队是否会在叛乱成功之后保留。在原版中，篡权者此项为yes
will_relocate = <boolean>#军队是否会在一块区域的目标省份全被占领后跨海进入另一片与之不相连区域的目标省份中。
gfx_type = [culture_province/culture_owner]#军队使用的图像效果，culture_province为省份本身的文化，culture_owner为省份拥有者的文化。

##军队模式
resilient = <boolean>#在士气降为0后是否消失。在原版中，篡权者此项为yes
reinforcing = <boolean>#是否能接收到增援。在原版中大多数叛军（农民除外）此项为yes
general = <boolean>#是否使用将军。在原版中大多数叛军（农民除外）此项为yes
smart = <boolean>#是否做出一些“智能举动”，例如从战斗中提前撤退等。在原版中大多数叛军（农民除外）此项为yes
infantry= <float>
cavalry= <float>
artillery =<float>#三种陆军单位占比。
morale =<float>#士气倍数

##可用的处理方式（显示在稳定与扩张界面“解决他们”中）
handle_action_negotiate = yes #同意请求
handle_action_stability = yes #提高稳定度
handle_action_build_core = yes #在目标省份造核
handle_action_send_missionary = yes #在目标省份传教
handle_action_change_culture = yes #改变目标省份的文化

##产生权重，在一个省份中，只有权重最高的类型才可能是本省爆发叛军的类型
spawn_chance = {
factor = <int> #基础权重
modifier ={
factor = <float> #权重修正值，如果为0则说明此叛军在满足条件时永远不会出现（事件除外）
<条件>
}

##移动方向权重
movement_evaluation = {
factor = 1
modifier = {
factor = <float>#满足条件时叛军的移向本处的权重
<条件>
}

##围城胜利带来的影响
siege_won_trigger = {
		<条件> #只有满足条件才可能进行siege_won_effect的判定
	}
siege_won_effect = {
custom_tooltip = #如果效果过长，可以使用自定提示框来缩减文本量，需要本地化。
hidden_effect = {效果} #同自定提示框对应的隐藏效果。
#或者直接在此处填写效果。
}

##强制同意要求相关
can_negotiate_trigger = {条件} #只有满足条件，才能在稳定与扩张界面中同意他们的要求。
can_enforce_trigger = {条件} #只有满足条件，叛军才能强制执行要求。一般填写always=yes
demands_description = "字符串" #字符串对应叛军要求的本地化
demands_enforced_effect = {效果} #强制执行要求给国家或者叛军所在省份带来的效果。
}#不要忘记最后的大括号


界面

需要在/Europa Universalis IV/gfx/flags中添加以叛军名称命名的.tga文件以创造叛军旗帜。

本地化

以下内容需要本地化。 

polish_noble_rebels_demand:0 "Return of Elective Monarchy" 
polish_noble_rebels_demand_desc:0 "The nobles demand we go back to having the old ways, where mounted cavaliers ruled estates of subservient peasants. They demand an end to this country of Merchants and Peasants."#对叛军要求及其描述的本地化。
polish_noble_rebels_title:0 "Magnates"
polish_noble_rebels_name:0 "$COUNTRY$ Magnate Rebels"#叛军名称
polish_noble_rebels_desc:0 "Magnate Rebels are dissatisfied with their monarch, usually due to liberal reforms. They field professional troops with a large proportion of cavalry."#叛军描述
polish_noble_rebels_army:0 "Noble Army"#叛军军队的名称

也要对custom_tooltip = 中的部分进行对应的本地化。
