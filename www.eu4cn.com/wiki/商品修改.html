增加商品类型

 此内容可能已落后版本，最后更新于1.25

商品文件的位置在 /Europa Universalis IV/common/tradegoods/。 所有的商品都要有独特的名称，并且按照如下格式编写： 

<TradeGood> = {
     #货物地图模式下的颜色
     color = { 0.96 0.93 0.58 }

     #控制货物贸易对国家带来的加成
     modifier = {
	land_forcelimit_modifier = 0.20
     }

     #生产本货物对省份提供的加成
     province = {
	land_forcelimit = 0.5
     }

     #货物在未殖民区域出现的概率
     chance = {
	factor = 35
        #You can specify it by adding modifiers so it's 32*1.5 = <Chance> if has_terrain = desert
	 modifier = {
                #factor = 0 代表永远不会出现
		factor = 0 
		area = newfoundland_area #触发条件
	 }
	 modifier = {     
		factor = 1.5
		OR = {
			has_terrain = desert #触发条件
			has_terrain = glacier #触发条件
		}
	 }
     }
}

商品价格

 此内容可能已落后版本，最后更新于1.25

价格的定义位置是 /Europa Universalis IV/common/prices/。 

<TradeGood> = {
        #基准价格
	base_price = 2.5 
        #可选，决定此货物是否使用黄金的机制，例如生产收入直接转化为收入，导致通货膨胀，不能建造加工厂类建筑，产量衰竭等等。
        goldtype = yes/no #Optional 
}

加工厂

 此内容可能已落后版本，最后更新于1.25

可以为货物指定加工厂。定义文件位于 /Europa Universalis IV/common/buildings/。 

<manufactory> = {
        # 在manufactory = {}中增添你的新货物。
	manufactory = {
		copper
		iron
                <TradeGood> 
	}
#是否在地图上显示模型
	onmap = yes	
#ai的建造倾向
	
	ai_will_do = {
		factor = 1
	}
}

给你的商品添加图片

 此内容可能已落后版本，最后更新于1.25

你需要两张图像给你的商品，一个28x28另一个64x64。 28x28 图像在 /Europa Universalis IV/gfx/interface/，文件名叫“resources_small”。然后是 64x64 图像在 /Europa Universalis IV/gfx/interface/，文件名叫“resources”。

你需要把该图像改宽 28/64 像素并且添加你的图像。 文件需要用 dds/tga 格式。

在界面里注册你的图片

 此内容可能已落后版本，最后更新于1.25

你需要到 /Europa Universalis IV/interface/ 然后在名为“provinceview.gfx”的文件里改变“noOfFrames”中的数字。 

spriteType = {
	name = "GFX_resource_icon"
	texturefile = "gfx//interface//resources.tga"
	noOfFrames = 33 # 在“resources”里的 64x64 图像的数量
	loadType = "INGAME"
}
	
spriteType = {
	name = "GFX_resource_icon_small"
	texturefile = "gfx//interface//resources_small.dds"
	noOfFrames = 33 # 在“resources_small”里的 28x28 图像的数量
	loadType = "INGAME"
}
