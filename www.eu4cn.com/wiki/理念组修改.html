理念组修改

本页面适用于最新的版本(1.36)。

这篇文章介绍对理念组的增添与修改，包括基本理念组和国家理念组。它们的文件位于/Europa Universalis IV/common/ideas。

理念组的通用结构

所有的理念组, 无论是基本理念组还是国家理念组，都要有7个理念和一个完成理念组后获得的额外奖励。它们可以任意命名，但是必需独一无二。

单个理念

单个理念应当包括名称和修正。举个例子： 

royal_navy = {
        navy_tradition = 0.25
        heavy_ship_power = 0.10
}

任何作用域为国家范围的修正都是可以使用的，同时理念名称也需要本地化。

额外奖励

完成理念组即可获得额外奖励，它所使用的修正前面所说的单个理念是相同的，不过不必进行本地化。 

bonus = {
        free_leader_pool = 1
}

触发条件

基本理念组和国家理念组都可以使用触发条件。对于基本理念组而言，触发条件可以决定哪类国家可以选择本理念组，如原版的贵族理念和财阀理念。国家理念组的触发条件则可以决定哪个(些)国家可以使用本理念。任何作用域为国家范围的触发条件均可。 

trigger = {
        OR = {
                tag = ENG
                tag = GBR
        }
}

基本理念组

基本理念组是由行政科技的升级所开启的。它们需要额外指定消耗哪类君主点数，例如军事类的： 

category = MIL

还要指定AI的开启倾向。
原版的基本理念组位于 00_basic_ideas.txt。新的基本理念组可以在本文件里加入，或者创建一个新的文件。

基本理念组的数目可以超过原版的数目，但是如果增加太多的话，你需要对界面进行修改，以充分显示。

AI倾向

可以为AI指定开启倾向。在下面的例子中，如果人文理念组开启，那么AI就不会选择本理念。 

ai_will_do = {
        factor = 1
        modifier = {
                factor = 0
                has_idea_group = humanist_ideas
        }
}

国家理念组

国家理念组在加载游戏时即被确定，并且可以用事件和决议中的“swap_free_idea_group”效果进行更新。一般这种情况只发生在改变国家tag的决议中。

国家理念组代码中要有“free = yes”这一行，用于和基本理念组进行区别。

原版的国家理念可以依次在00_country_ideas.txt, zz_group_ideas.txt,和zzz_default_idea.txt在找到。新的理念可以在这些文件中添加，或者创建一个新的。这种情况下，就要注意理念组的加载顺序（见下）。

传统

国家理念在开头处添加如下代码，以对应“传统”。它们同样是作用域为国家范围的修正。示例如下： 

start = {
        naval_morale = 0.20
        embargo_efficiency = 0.33
}

国家理念组的序列

如果一个国家满足多个理念组的加载需求，它永远是加载最靠前的那一个。理念文件的读取按照字母顺序进行，因此00_country_ideas.txt（独特国家理念）中的文件总是先于zz_group_ideas.txt（个别国家/科技组/政体/政府改革/文化组理念），后者又比zzz_default_ideas（通用国家理念）靠前。而在单个文件内部，是按照理念的出现顺序进行分配。这种情况是为了防止一个本来可以使用独特国家理念或者文化组理念的国家，使用了通用国家理念。在新增国家理念时也要注意这些。

举个例子： 冰岛在原版中使用通用国家理念，是因为独特国家理念和文化组理念的文件中没有给它一个特别的理念。在zz_group_ideas.txt中添加一个触发判定为trigger = { culture_group = scandinavian }的新理念组后， 冰岛就可以使用本理念组而非通用国家理念，而 瑞典和 丹麦的独特国家理念不受影响。但是如果你错误地将这个新理念组加到00_country_ideas.txt的第一位，那么 瑞典和 丹麦的独特国家理念也将被覆盖。

本地化

理念组的名称、单个理念的名称与介绍，以及国家理念的传统和额外奖励（野心）都应被本地化。具体如下： 

KHM_ideas: "Khmer Ideas"
KHM_ideas_start: "Khmer tradition"
KHM_ideas_bonus: "Khmer ideas"
staunch_theravada_buddhists_desc: "The Khmer people have fully embraced Theravada Buddhism as a reaction to the decadence of the past"
staunch_theravada_buddhists: "Staunch Theravada Buddhists"

首先对理念组的名称本地化，"[组名]_start" 是国家理念的传统， "[组名]_bonus" 是国家理念的额外奖励（野心）。单一理念的名称直接引用，而它的介绍则是在 "[理念名]_desc"后。
