1.27版本，代号“波兰”，于2018月10月1日[1] 发布，校验码为edf9。

免费特性

  • 可支持Steam好友的动态细节信息（Steam Rich Presence）
  • 新的抽签选举机制。
  • 新加入威尼斯独有的政体（威尼斯尊贵共和国）。
  • 新的第三级抽签政府改革可启用抽签选举制（达摩）。
  • 添加共和国可通过委任政府改革启用的科层制机构和委员会任命制（达摩）。
  • 移除旧版抽签改革（达摩）。
  • 添加卡尔梅克人到来的事件。
  • 添加普鲁士联盟事件链。
  • 添加摩尔达维亚的罗马人事件链。
  • 添加波兰、立陶宛、罗马尼亚诸国、鞑靼诸部落的任务树。
  • 添加成立伊尔汗国的决议。
  • 为蒙古帝国添加大蒙古国政体（免费），允许在主流文化的省份招募旗军（天命）。
  • 为拥有马拉塔阶层的印度国家添加新的马拉塔常驻咨议会政府改革。给予5%训练度，并将招募顾问的选项换为征召一个＋1冲击的将领，而是不是招募一位顾问（达摩）。
  • 为拥有拉杰普特阶层的印度国家添加新的普尔比亚政府名册的政府改革。将招募普尔比亚这一互动变为给拉杰普特兵团提供+5%训练度的加成（达摩）。

游戏平衡

经济

  • 若国家高度负债，则禁用购买特许公司，以避免通过这种机制在其它国家存钱。
  • 关税现在只计入生产、黄金和贸易的收入。
  • 宗主国收到的关税现在和殖民地国家实际支付的数额相匹配了。

政府

  • 为共和国添加缺少的最大专制度。
  • 将夺取行政权改革造成的改革损失从4项增加至7项。
  • 为宣告圣神指引改革造成的改革损失从6项增加至8项。
  • 选举中的亲信候选人现在随机年龄在30到60之间。
  • 将成立共和国改革造成的改革损失从6项降低至3项。
  • 部落文明社会改革现在会获得额外-0.05年腐败度变化。
  • 部落空头许诺改革修正减少至+10%直属州维护费修正。
  • 部落宗教社团改革现在额外提供+15%宗教统一。
  • 大主教区无法再让其君主成为将领，而神权政体则可以任意让其君主成为将领。各个大团长已经开始准备反击啦。

部队

  • 拉杰普特军团基础修正修改为-25%专业化衰减速度、-50%的增援花费和+5%士气。

其他

  • 为来自当贵族阶层的本地继承人每月增加0.33支持度（选举君主制）。
  • 将教宗政体的年度威望增加改为以发展度衡量的来自传教士的威望。
  • 将教宗国理念罗马的辉煌（+1年度威望）更换为-1%威望衰减。
  • 将教宗国理的梵蒂冈博物馆（-1%威望衰减）更换为-10%外交顾问花费（＋1免费外交政策-达摩玩家）。
  • 梵蒂冈图书馆事件的行政选项会-5%行政科技花费。
  • 梵蒂冈图书馆事件的外交选项会-5%外交科技花费
  • 梵蒂冈图书馆事件的军事选项会-5%军事科技花费，并增加在战斗中获得额外50%威望的效果。
  • 梵蒂冈图书馆事件的军事选项不再给予5%训练。
  • 检测到污染，开始隔离协议。
  • 毗奢耶那伽罗的任务“从孟加拉转移贸易”(将4个地区的贸易中心升一级)对2级贸易中心也将起作用。
  • 世界大港给予-0.2%的海军传统衰减，而不是+0.25年度海军传统。
  • 宗教理念全部采用后可以让你对自治领传教。
  • 自治领有-2%传教力量惩罚。
  • 齐米阶级控制的省份对全局的宗教统一现在影响更小，当他们处于中立时会减少50%，处于忠诚时减少100%。
  • 不会再从市场，贸易站，证券交易所获得全球贸易思潮，因为玩家已经从贸易力量以及这些建筑自带的加成中得到很多增益了。
  • 拥有3级贸易中心的省份会获得0.5的全球贸易每月思潮接纳加成，2级贸易中心会提供0.1的思潮接纳加成。
  • 将在一定程度上限制工厂生产思潮产出的地点，这样就不会让它出现在5发展度的沙漠省份。
  • 降低在非发源大洲上工厂生产思潮的传播速度。
  • 将在一定程度上限制启蒙主义思潮产出的地点，这样就不会让它出现在5发展度的沙漠省份。
  • 降低在非发源大洲上启蒙主义思潮的传播速度。

AI

经济

  • 修复了AI会在征召军队时贷不必要的款。
  • 修复了AI不会在超出海军上限时解散舰队。
  • 修复了AI向玩家提出特许贸易公司请求时不会给予玩家金钱的问题。

战争

  • 在非流亡部队重新规划路径时，会考虑堡垒控制区。
  • 如果AI是防守方并且已经提交了和平提议，围攻战争目标省份时不会给予任何惩罚。

其他

  • 修复了AI在规划以及进行殖民/定居时的不确定性。

界面

国家

  • 在附属国界面中，点击贸易公司上的盾标可以聚焦到那个贸易公司的地图上。
  • 议会议案现在会有一个滚动条，这样你可以通过滚条看到议案的全部部分。

图标/美术

  • 修改了性别图标，让它看起来更清晰。

其他

  • 可以在本地化文件中找到流亡君主相关内容了
  • 可以定义统治者的年龄范围
  • 可以标记一个政府改革，在统治者死后触发选举。

用户制作方面

修正

  • Can no longer set dynasty on republican leaders in define_ruler effects. Dynasty argument is reused to let you set last name now.
  • Added add_next_institution_embracement_scaled which behaves like devving in a province
  • Added adm_power_cost, dip_power_cost, mil_power_cost effects & triggers that scales with the power cost modifier
  • All Variable effects can be defined with `= { var_name = value }` now.
  • Added set_base_tax, set_base_production, set_base_manpower effects

Modifiers

  • Implemented monthly_support_heir_gain modifier
  • Added new modifier that increases how much PP you get from an insult.
  • has_banners modifier now will work for every country and is no more limited to Manchu
  • Added static modifier that is applied to all territory provinces.
  • Added local_religious_unity_contribution modifier.

触发器

  • Added has_unembraced_institution trigger.
  • All Variable triggers can be defined with `= { var_name = value }` now.

其他

  • Can now access exiled monarchs in localisation
  • Can now define the age range of a ruler
  • Can now flag a government reform that it triggers elections upon death.

脚本

决议

  • 增加成立蒙古帝国的决议
  • 增加重建伊利汗国为卫戍国的决议
  • 将首都迁至圣彼得堡将不会再升级它的贸易中心等级。

事件

  • 在“文化纽带衰弱”事件中，增加一个可以将该省份转换为美洲主流文化的选项。第一个选项会将该地区所在的美洲文化接纳为可接纳文化。
  • 重写了“穿刺公弗拉德三世”事件。弗拉德现在会更频繁地出任君主。

理念

  • 增加切尔尼科夫国家理念。
  • 增加克里米亚国家理念。
  • 增加伊尔汗国国家理念。
  • 为动态哥萨克国家增加国家理念。
  • 增加里加国家理念。

设定

  • 增加蒙古帝国以及卡尔梅克的国家tag。
  • 波兰会在1444年接受鲁塞尼亚文化

其他

  • 在自定义理念中增加新的“免费政策”修正。
  • 增加瓦拉几亚和摩尔达维亚的开局历史背景说明。
  • 增加鞑靼游牧诸部的开局历史背景说明。
  • 河中的蓝色地图切换为另一种不同的蓝色。
