难度

不同的“难度”设置给予玩家或AI国家不同的加成，以调整游戏的整体难度。如果设置为正常，没有国家会得到加成。在简单的情况下，AI对玩家更宽容，而在困难的情况下，AI则更具攻击性。

幸运国家

决定是否有8个历史，或者8个随机，或者没有幸运国家。

在地图上表示未知之地

开启或关闭战争迷雾。特别是在你自己发现前不想提前知道随机新大陆长什么样时很有效。铁人模式不可用。

使用动态省份名称

当省份的文化组或者拥有者改变时，允许或禁止更改省份的名称。如果该选项关闭，只能人为改名，系统不能对其进行改名。

显示月度税收

在地图上以动画文本的形式显示月度税收。

“有主”荒土上色

荒芜省份将被涂上周围主要控制者的颜色。

飞地使用地区名

飞地命名使用地区名（如以“瑞典刚果”代替“瑞典”。）

理念组选择没有限制

你可以选择任何的理念组，不再受类别的限制。

幻想的随机新大陆

控制在随机新大陆出现幻想元素（如高度发达的原住民帝国）的几率。

显示开始界面

开关是否在一局新游戏开始时显示有关你的国家与所在地区的背景信息的开始界面。

多人游戏

 此内容可能已落后版本，最后更新于1.29

允许玩家热加入

如果开启，玩家可以加入一场已经开始的游戏，使其变成多人游戏。在铁人模式启用时不可用。

允许多个玩家使用同一国家

如果开启，多个玩家可以控制同一个国家。在铁人模式启用时不可用。

仅允许主机与观察者存档

如果开启，只有主机和观察者将能保存多人游戏的存档。I

仅允许主机暂停

如果开启，只有主机能够暂停游戏。

Editable savegame

If on, players can manually edit saved game files between sessions.

Locked ledger

If on, players will not be able to open the ledger, preventing usage of it for seeing other countries' armies.

Limited ledger

If on, the amout of information given by the ledger is reduced.

Use victory cards

Enables od disables victory cards in the game.

Use age scoring

If on, scores and victory cards will be reset with every new age.

Use teams

If on, members in a team can only ally each other and not other players.

Free team creation

If on, every player can create a team in the lobby.

Prevent nation ruining

Blocks certain destructive actions by players that have lost a war as long as the truces after that war are in effect.

Allow province name change

Allows players to change names of provinces.

自订设置

 此内容可能已落后版本，最后更新于1.29
只适用于DLC黄金国激活时。

自定义国家难度

这个选项决定玩家创建自定义国家时的可用点数。点数越多，难度越低。

点数: 50 - 100 - 200 - 400 - 800

省份基税及人力

这个选项决定所有省份的基税和人力。

正常值:

所有省份具有史实的税收和人力值。

随机值:

所有省份具有随机的税收和人力值。

同一值:

所有省份具有相同的税收和人力值。(4-4-3)

国家

这个选项决定地图上会出现哪些国家。

正常设置:

启用自定义国家，这张地图将由史实国家和已启用的自定义国家组成。

史实设置:

禁用自定义国家，这张地图仅由史实国家组成。

随机设置:

这张地图将由玩家自定义的国家和随机生成的国家组成。

自订设置:

这张地图仅由玩家自定义的国家组成，其他省份都是未殖民省份。所有国家开局获得一个殖民者。

允许向存档中加入国家

允许向存档中加入自定义国家。

随机国家

这个选项决定了随机自定义国家的名称及旗帜样式。

动态:

随机自定义国家会得到动态生成的名字和旗帜。

史实:

随机自定义国家会得到历史上存在的国家的名字和旗帜。
