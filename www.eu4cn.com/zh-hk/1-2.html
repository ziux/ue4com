擴展包特性

  •  鄂圖曼：新增含有68項任務的新任務樹、2種特殊機制、12個新政府改革、新的耶尼切里階層（附帶有一系列新階層特權、階層議程和2種災難）、5個新災難、62個新事件、新附屬國類型：行省，以及2種修會。
  • 中國地區：新增5種新任務樹（ 大明、 建州女真、 滿洲、 大清、中國群雄、中國皇帝，總計77項新任務）、2個新政府改革、新的宦官階層（附帶有一系列新階層特權、階層議程和1個宦官奪權的災難）、32個新事件、巡撫三司機制（使用修會機制），以及1種新直屬州法令。
  •  日本：新增含有30項任務的新任務樹（符合條件時亦可獲得中國皇帝任務樹）、新政府機制（基督教日本可用）、6個新政府改革、1個新階層特權、12個新事件，以及2個特有宣戰理由。
  •  俄羅斯：為 莫斯科和 俄羅斯新增含有57項任務的新任務樹，為 諾夫哥羅德新增7項新任務；為俄羅斯新增2種特殊機制（現代化和官秩表）、9個新政府改革、12個新階層特權、65個新事件（含原有事件重做），以及2個新決議。
  •  西班牙：擴展並重做了 卡斯蒂利亞、 阿拉貢和 西班牙的任務樹，新增20項新任務，新增1種特殊機制（諮議會行政制）、3個新政府改革、2個新階層特權、1個災難，以及7個新事件。
  •  法蘭西：新增含有68項任務的新任務樹以及革命法蘭西專屬的16項額外任務、4個新政府改革、24個新事件，和新增的附屬國類型：封邑。
  •  大不列顛：為大不列顛新增了含有71項任務的新任務樹、4個新政府改革、41個新事件、超過40項特殊議會相關改動、2種新附屬國類型（商貿公司和貿易保護國），並重做了英國內戰災難。
  •  普魯士：重做了普魯士任務樹，新增了部分觸發條件和獎勵，添加了新機制（普魯士軍事化）和6個新事件。
  •  葡萄牙：重做了葡萄牙任務樹，新增了5個新任務，新增了部分觸發條件和獎勵，以及2個新政府改革。
  •  朝鮮：新增了特殊政府機制（「至善論」），可獲得中國皇帝任務樹，新增2個新政府改革，以及「高麗大藏經」相關事件和省份修正。
  • 新增了68個特有政府改革，分別適用於不同國家和文化組。
  • 新增了60個階層特權，分別適用於各個階層。每個階層平均增加了5個新特權。
  • 新增了9種特殊單位，其中3種為陸軍單位（ 日本專屬的「武士」、 西班牙專屬的「大方陣」和 法蘭西專屬的「火槍手」），6種為海軍單位（伊比利亞文化專屬的「卡拉維爾帆船」和「蓋倫帆船」、拉丁文化專屬的「加萊賽戰船」、 朝鮮專屬的「龜船」、 大不列顛專屬的「主力艦」Man-of-war和 尼德蘭專屬的「東印度公司商船」）。
  • 新增2個兵模包，一個用於法蘭西文化組文化（法蘭西、奧克、瓦隆和勃艮第文化小國）、一個用於東方文化（中華文化組南方小國、阿伊努、大理和菲律賓）。
  • 新增3個音樂包，分別用於法蘭西、鄂圖曼和中國，以及DLC預購獎勵的「March of Power」音樂包。

遊戲平衡

經濟

  • 如果宗主國進行了相應改革但其附屬國沒有，則「擴大教會權利」不再適用於附屬國。
  • 擴展基礎設施現在將會使得發展成本降低 15% 而不是以往的 25%。
  • 重商主義效果增強。當重商主義為 100 時，禁運效率會從 50% 提高到 100%，市民/吠舍的忠誠均衡點會從 5% 提高到 10%。
  • 省份叛亂度現在每有 1 點，將減少 2% 的省份稅收修正。
  • 省份稅收現在將會提供 -2% 本地軍團招募時間、-1% 本地偉大工程升級時間、-1% 本地建設時間和 +2% 本地思潮傳播。
  • 通貨膨脹現在也增加了傳教士維護、升級偉大工程、接納機構、調整殖民附屬國、升級貿易中心和貿易公司投資的成本。
  • 將 法爾斯的顏色更改為薄荷綠。

政府

  • 議會席位現在會給予省份+10%本地水手修正與+15%本地人力修正，而生產效率與人力不變。
  • 給予一個20發展度的省份議會席位現在會減少2點專制度，而不是3點，無論席位是自動給予的或是手動給予的。
  • 貿易聯盟的每一個成員現在都會給予聯盟領袖+5%貿易引導、+2海軍上限、+1000水手上限、+1000人力上限以及+1陸軍上限。
  • 現在擁有軍事化機制時基礎給予-0.5每月軍事化，在戰爭中會給予+0.1每月軍事化，在和平中會給予-0.1每月軍事化，每點厭戰度也會給予每月軍事化減成。
  • 謝契拉達政府改革現在解鎖游牧理念，並獎勵 25% 特殊單位上限。
  • 謝契拉達現在可以使用階層機制。其中包含哥薩克階層。
  • 「重選權正式化」政府改革現在與「尼德蘭共和國」互斥，因為後者不允許強制重新選舉。
  • 原住民國家無法再從政府改革中獲得陸軍維護費修正、全局部隊花費和人力恢復速度方面的加成。
  • 天主教普魯士或自定義國家現在可以獲得普魯士君主制的1級政府改革。
  • 「地方議會」政府改革解鎖的直屬州法令「宗教寬容法令」現在給予的本地宗教統一度貢獻從 50% 調整至 100%。
  • 「維持權力平衡」政府改革獎勵的所有階層忠誠度從 5% 增加至 10%，並給予 +1 外交官。

宗教

  • 新教教會信條「教牧人員允許結婚」現在給予 -15% 同一宗教顧問花費，不再減少教士階層影響力。
  • 宣戰現在不再對虔誠度產生影響。新統治者繼位時虔誠度改為刷新為 0。
  • 負虔誠度現在給予 -10% 理念花費，不再獎勵 20% 要塞防禦。
  • 貿易公司減少的本地傳教力量從 200% 調整至 20%。
  • 教廷監護現在給予「派遣教廷使節」的獎勵。
  • 科普特正教給予的要塞防禦從10%調整至15%。所有佛教信仰給予的異端容忍從+1調整至+2。神道教給予的+1異教容忍調整為+2異端容忍。
  • 重製了改革宗熱情機制。啟用一個熱情焦點的花費和維護費用從8改動至5。熱情點現在給予-10%建築花費、-10%穩定度花費、-5分離主義年數、+10%改善關係修正。 穩定焦點給予的-10%建築花費改為-25%間諜行動花費。新增了「外交」熱情焦點，獎勵+2外交聲譽、+25%改善關係修正、+10%所有階層忠誠度均衡點。

單位

  • 哥薩克特有的+10%衝擊傷害增至+15%。
  • 恰瓦花費的軍事點數從5調整為2，新招募時改為具有50%初始兵力、10%士氣。恰瓦軍團現在具有-5%所受衝擊傷害、-50%陸軍損耗、+50%增援速度。
  • 耶尼切里新招募時改為具有10%初始兵力、10%士氣。特殊修正從+100%操練度獲取修正、+100%補員花費改為+50%強攻要塞能力、+50%操練度獲取修正。
  • 哥薩克單位現在不再給予提升穩定度花費懲罰。
  • 射擊軍現在不再給予提升穩定度花費懲罰，且獲得+10%火力傷害、-10%所受火力傷害、+15%陸軍維護費。

戰爭與和平

  • 荒廢度給予的懲罰加倍。
  • 被封鎖的省份獲得的本地貿易力量懲罰從-50%增至-100%。修建市場建築將無法再完全抵消這一懲罰。
  • 呼籲和平給予的厭戰度從0.008調整至0.005。
  • 重新平衡單位點數。削弱澳洲原住民和波利尼西亞兵種組單位，修正非洲、穆斯林兵種組單位隨科技提升反而變弱的問題，削弱安納托利亞兵種組的早期單位，對部分印度兵種組單位進行平衡。
  • 「鼓舞人心的將領」將領特質現在除士氣恢復速度之外，還獎勵+5%陸軍士氣。
  • 「生於馬背之上」將領特質現在給予+100%騎兵比例。刷出該特質所需的騎兵比例從20%調整至30%。

其它

  • 改變宿敵的冷卻年限從25年改為5年。
  • 每1點原住民侵略性現在降低1%本地殖民幾率。
  • 熱帶省份給予的本地發展花費懲罰從10%調整至5%。

AI

外交

  • 在會由於stability_cost_to_declare_war失去穩定度時，AI不會再宣戰。

其它

  • AI使用的炮兵將減少，尤其是收入低時。

  • 修復了部分情況下AI軍隊因懼怕敵軍而不行動的問題。

用戶模組製作

命令

    • 添加了命令 guibounds，允許在啟用 UI mod 時調試 UI 元素。
    • 添加了 set_country_flag 和 clr_country_flag。

效果

  • 添加了 change_country_color = { country = <country scope> / color = { r g b } } ，如果使用前一個參數，那麼國家將變為指定範圍的顏色。如果使用後者，則將使用定義的顏色。當然，也有 restore_country_color = yes。
